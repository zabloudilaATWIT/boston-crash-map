Senior Project

Interactive Mapping Analysis

Project Overview

This interactive map demonstrates our research findings and geographical analysis for our senior project.

Key Features

Explore different layers, markers, and data visualizations directly on the embedded Felt map.

How to Use

Click and drag to pan, scroll to zoom, and click on markers to view detailed information about each location.

Ask About Crashes in an Area

Type any question you have about crashes in a street, neighborhood, or area. For example: “Are there many crashes near Fenway?” or “Which intersections on Mass Ave are the riskiest?”
